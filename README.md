# 2048

Vlastní varianta hry 2048. **[Hrát online →](https://zirafka.github.io/2048/)**

## Jak se hraje

Posouvej dlaždice šipkami (nebo W/A/S/D, na mobilu swipem). Když se dvě stejné dlaždice srazí, sloučí se. Cílem je sestavit na hrací ploše všech 16 mocnin dvou – od 2 do 65 536 – každou právě jednou.

## Odlišnosti od klasického 2048

- Výherní podmínka: všech 16 unikátních mocnin na ploše naráz (ne jen dlaždice 2048).
- Neomezené undo (tlačítko Zpět nebo Ctrl+Z).
- Černobílý vizuální styl – dlaždice se liší hustotou šrafování, ne barvou.
- Cloudové ukládání stavu přes JSONBin.io.

## Technologie

Vanilla HTML + CSS + JavaScript, žádné závislosti, žádný build step.
